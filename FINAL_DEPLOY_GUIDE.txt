 طريقة النشر السريعة (الأسهل - مجانية):
══════════════════════════════════════════════════════════════════════

1️⃣ أنشئ قاعدة بيانات Supabase (مجاني):
   → افتح: https://supabase.com
   → سجّل دخول (Google/GitHub)
   → New Project → الاسم: royal-pipes
   → انتظر دقيقة

2️ أنشئ الجداول:
   → اذهب إلى: SQL Editor → New query
   → انسخ الكود من SETUP_GUIDE.txt والصقه
   → اضغط Run ✅

3️⃣ احصل على المفاتيح:
   → Settings (⚙️) → API
   → انسخ: Project URL و anon/public key

4️⃣ ارفع الموقع:
   → افتح: https://app.netlify.com/drop
   → اسحب مجلد dist/ وأفلته
   → ✅ الموقع الآن على الإنترنت!

5️⃣ اربط الموقع بالسحابة:
   → افتح: https://موقعك.netlify.app/admin
   → سجّل دخول: admin / royal2026
   → اذهب إلى "الإعدادات"
   → الصق Project URL و anon key
   → اضغط "اختبار الاتصال"
   → ✅ الآن المنتجات تُنشر للجميع!

📁 جذر المشروع:
  ├── dist/                    ← الموقع المبنى (جاهز للرفع على Netlify)
  ├── api/                     ← خادم Node.js + MySQL API
  │   ├── server.js            ← API Server (Express + MySQL)
  │   ├── setup-database.js    ← إنشاء الجداول + البيانات الافتراضية
  │   ├── package.json         ← تبعيات Node.js
  │   ├── .env.example         ← نموذج ملف الإعدادات
  │   └── deploy.sh            ← سكربت النشر السريع (اختياري)
  ├── src/
  │   ├── lib/
  │   │   └── mysql-api.ts     ← ربط React مع API الخادم
  │   ├── data/
  │   │   ── store.ts         ← مخزن البيانات (localStorage)
  │   ├── pages/               ← صفحات الموقع
  │   ├── components/          ← المكونات المشتركة
  │   ├── router.tsx           ← نظام التوجيه
  │   └── App.tsx              ← المكون الرئيسي
  ├── README.txt               ← دليل الاستضافة السحابية (Supabase)
  ├── SETUP_GUIDE.txt          ← دليل مفصل مع كود SQL
  ├── DEPLOYMENT_GUIDE.txt     ← دليل النشر على VPS + MySQL
  └── FILES_SUMMARY.txt        ← هذا الملف

══════════════════════════════════════════════════════════════════════
 طريقة النشر 1: سحابية بسيطة (Supabase + Netlify)
══════════════════════════════════════════════════════════════════════

  ✅ الأسهل - لا تحتاج خادم
  ✅ مجاني بالكامل
  ✅ الخطوات في README.txt

  1. أنشئ قاعدة بيانات على Supabase
  2. ارفع مجلد dist/ على Netlify
  3. أدخل مفاتيح Supabase من لوحة التحكم

══════════════════════════════════════════════════════════════════════
 طريقة النشر 2: خادم حقيقي (VPS + MySQL + Node.js)
══════════════════════════════════════════════════════════════════════

  ✅ تحكم كامل بالبيانات
  ✅ أسرع وأقوى
  ✅ الخطوات في DEPLOYMENT_GUIDE.txt

  1. اشترِ VPS (DigitalOcean ~$6/شهر)
  2. ثبّت Node.js + MySQL + Nginx
  3. أنشئ قاعدة البيانات
  4. شغّل API الخادم
  5. ارفع مجلد dist/
